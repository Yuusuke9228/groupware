## Style Rules

- 英語UIは sentence case を基本とする。
- ボタンは短い動詞を優先する（`Save`, `Create`, `Run backup`, `Download`）。
- 同一概念に複数訳を使わない。
- help / admin-manual でも UI 表記と同じ語を使う。
- バリデーションは命令形ではなく利用者に分かる文にする（例: `Please enter a title.`）。
