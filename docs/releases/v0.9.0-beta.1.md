# TeamSpace v0.9.0-beta.1

## 日本語
- 初回の公開リリース候補です。
- スケジュール、メッセージ、ワークフロー、タスク、日報、掲示板、WEBデータベース、ファイル管理などの主要機能を含みます。
- PWA、通知、Google Calendar / iCal 連携、OIDC / SAML / SCIM に対応しています。
- 今後、英語UI、多言語対応、ドキュメント強化、スクリーンショット追加を進めます。

## English
- This is the first public release candidate of TeamSpace.
- Includes core features such as schedule management, messaging, workflow, task management, daily reports, bulletin board, web database, and file management.
- Supports PWA, notifications, Google Calendar / iCal integration, and enterprise auth including OIDC, SAML, and SCIM.
- Upcoming improvements include English UI, multilingual support, stronger docs, and additional screenshots.
