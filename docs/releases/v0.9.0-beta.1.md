# v0.9.0-beta.1 Release Notes

## English

### TeamSpace v0.9.0-beta.1 (Pre-release)

This is the first public beta release positioned for broader OSS discovery.

TeamSpace is a self-hosted groupware platform built with PHP/MySQL for teams that want to run internal collaboration tools on their own infrastructure.

### What you can use today

- Schedule management (day/week/month + organization views)
- Workflow approvals (templates, routing, dashboard, export)
- Kanban-style task management (personal/team/organization boards)
- Daily reports (structured entries, analysis fields, CSV export)
- Web database module (custom fields, relations, views, aggregation/chart basics)
- Messaging, bulletin board, notifications, file management, facilities, address book
- PWA baseline and enterprise auth foundations (OIDC/SAML, SCIM endpoints)

### Demo / Docs

- Live demo: https://groupware.yuus-program.com/
- User help: https://groupware.yuus-program.com/help
- Admin manual: https://groupware.yuus-program.com/help/admin-manual

### Scope of this beta tag

- Public OSS launch metadata optimization for GitHub
- First beta release channel setup (`v0.9.0-beta.1`)
- Repository docs alignment for release communication

### Known limitations

- Primary UI/docs language is currently Japanese-first.
- Multilingual UX and documentation are planned, but not fully implemented yet.

### Roadmap direction (planned)

- Stronger multilingual support
- Better self-host deployment experience
- Community-driven hardening and extensibility
- Optional commercial paths (implementation support / managed cloud offering)

### Feedback

Issues, discussions, and contribution PRs are welcome.

## 日本語

### TeamSpace v0.9.0-beta.1（プレリリース）

本リリースは、TeamSpace を海外も含めた OSS 利用者に伝えるための初回公開ベータです。

TeamSpace は PHP/MySQL で構築されたセルフホスト型グループウェアで、社内業務の情報共有・申請・タスク・データ管理を一元化できます。

### 現在利用できる主な機能

- スケジュール管理（日/週/月 + 組織表示）
- ワークフロー（テンプレート、承認経路、ダッシュボード、エクスポート）
- カンバン型タスク管理（個人/チーム/組織ボード）
- 日報（構造化入力、分析項目、CSV出力）
- WEBデータベース（カスタム項目、リレーション、ビュー、集計/グラフ基礎）
- メッセージ、掲示板、通知、ファイル管理、施設予約、アドレス帳
- PWA基盤、企業向け認証基盤（OIDC/SAML、SCIMエンドポイント）
